This project is about locating the position of drawing objects. Positioning data is given as command-line arguments to the Python script. The Python script produces output based on command line arguments (which are the position data of drawing objects). The idea behind this project was that if the user draws something, the project will be able to connect that drawing object to the real world.

Disadvantages:
● This app cannot recognize any object due to the missing python module.
● Python script only works in unity 3d editor..
