֎ SDMR (Shoot and Drive with Mind Reader) is built in C-Language. There are only 3 levels in this game.


● Level 1 is about the shooting. At this level, the user has only 10 bullets to shoot the enemy. If the user is able to shoot the enemy then the user will pass level 1 and reach level 2.

● Level 2 is about racing (not the actual race), but the user must protect their bus against CPU buses and the entire background environment will also move in the opposite direction to the user's bus. Level 2 is passed by the user only when the user can access a specific location and the user bus will not be strict with any other bus.

● Level 3, in this level, the user is locked in a room but the door will open only when the user enters the correct password into the input box. Therefore, the user has to find the password to avoid objects from this room. If the user will be able to find the correct password and enter the password then S / he will pass the last level.
