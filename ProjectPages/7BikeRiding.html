Bike Riding



SELF Work (Offsite)

Jul 2019 to Sep 2019

This project simulates the bike riding concept.

Download


This project simulates the bike riding concept. this project contains all the features which a bike contained like gear, clutch, etc. Users can control this bike by gear buttons, accelerator button, break button. To turn left /right user can tilt the phone left/right.

Disadvantages
The bike starts in 4th gear not in 1st gear.
