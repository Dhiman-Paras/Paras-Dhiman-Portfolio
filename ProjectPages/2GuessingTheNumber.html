This game is all about random guesses. User guesses an integer random number from 0 to 99. Guessing The Number game is only for Android users.

Install From Play Store


This app is a simple interesting fun game app. This app is specially designed for kids under 6-9 age and above. This game is all about random guesses of player. In this app users can type a random number between 0 to 100 and then s/he have to check his/her guess. If the guess is correct then users will be able to distinguish between numbers. This app contains no ads. The Number app is totally free for everyone.
